ReFormX

ReFormX 动态表单生成器

基于 React 和 Ant Design 的现代化表单解决方案，让表单配置变得简单而强大

常用表单组件

支持输入框、选择器、日期选择等多种常用表单组件

灵活布局

支持行内、网格、标签页、折叠面板等多种布局方式

表单联动

强大的表单项联动功能，支持复杂的业务逻辑

异步数据处理

支持异步数据加载和处理，轻松对接后端接口

自定义表单项

支持自定义表单项类型，满足特殊业务需求

表单验证

内置完善的表单验证机制，支持自定义验证规则

快速开始

安装

npm install reformx

基础使用

    import { ReFormX } from 'reformx';
    
    const App = () => {
      return (
        <ReFormX
          schema={{
            type: 'object',
            properties: {
              name: {
                type: 'string',
                title: '姓名'
              }
            }
          }}
        />
      );
    };
